## Objetivo unico

Vender el dominio. El sitio no debe presentar el dominio como un negocio operativo, agencia activa, servicio vigente ni entidad asociada a terceros. Debe dejar claro que es un activo digital premium en venta.

## Requisitos de diseno

- Estilo 2026: mucho espacio en blanco, layout limpio, tarjetas premium, sombras suaves y microinteracciones sutiles.
- Colores: azul profundo `#0F172A`, azul corporativo `#1E40AF`, verde confianza `#10B981`, blanco y grises slate.
- Tipografia: Inter o system UI.
- Mobile-first, responsive y ultra rapido.
- Sin JavaScript innecesario. Usar HTML5 semantico y Tailwind CSS listo para produccion.

## Estructura obligatoria

1. Hero section potente:
   - Headline: **"[DOMINIO_PREMIUM] - El Dominio Perfecto para [COMPRADOR_IDEAL]"**
   - Subheadline: explicar por que el nombre tiene intencion comercial, memorabilidad y relevancia en el sector.
   - CTA principal: **"Comprar este dominio"** o **"Contactar por WhatsApp"**
   - Trust: **".com premium - Keyword exacto - Listo para usar"**

2. Por que este dominio es valioso:
   - Valor semantico del nombre.
   - Relevancia actual del sector.
   - Ventajas de branding, SEO y posicionamiento.
   - Comparacion con dominios genericos, largos o con guiones.

3. Oportunidades de uso:
   - Crear tarjetas para casos reales:
     - [CASO_USO_1]
     - [CASO_USO_2]
     - [CASO_USO_3]
     - [CASO_USO_4]
     - [CASO_USO_5]

4. Ventajas del dominio:
   - Keyword exacto de alta intencion.
   - Facil de recordar y pronunciar.
   - Potencial SEO fuerte.
   - Profesional y creible.
   - Listo para captar leads o lanzar marca.

5. Prueba social:
   - Usar testimonios generales sobre valor de dominios premium, naming, conversion y confianza.
   - No inventar clientes reales del dominio.

6. FAQ de venta de dominios:
   - Precio.
   - Transferencia.
   - Escrow.
   - Por que vale la pena.
   - Si el dominio representa o no un negocio activo.

7. Precio y compra:
   - Mostrar **[PRECIO_BIN_O_CONSULTAR]**.
   - Incluir CTA a WhatsApp: `https://wa.me/[WHATSAPP_CON_PREFIJO_PAIS]?text=Hola%2C%20quiero%20comprar%20[DOMINIO_PREMIUM]`
   - Incluir CTA a email.
   - Explicar transferencia segura.

8. Footer:
   - Disclaimer claro:
     **"[DOMINIO_PREMIUM] es un dominio premium en venta. Este sitio solo muestra el valor del dominio y no representa ninguna agencia operativa ni servicio activo."**
   - Datos de contacto del vendedor.
